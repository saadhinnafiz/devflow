# DevFlow

A full-stack Q&A platform inspired by Stack Overflow, built with Next.js 16, TypeScript, and Tailwind CSS.

## Features

- Question and answer system
- User authentication
- Voting and reputation system
- Tags and categories
- Search and filtering
- AI integration

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- MongoDB

## Getting Started

1. Clone the repo
2. Install dependencies: `npm install`
3. Run the dev server: `npm run dev`
